# 神奇的代码注释

## 说一些东西

因为 java 的注释只在开头结尾就可以，比较简洁，所以这里采用 html 进行注释。

复制源码后放在想要放的位置，一般编辑器一键注释的快捷键是 `Ctrl+/`，不会就自己手动添加。

因为大部分编辑器都有拼写检查，所有有些注释中的波浪线会让人不舒服，各位自行选用。
